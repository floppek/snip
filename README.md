# snip

A free and simple way to upload and share files.

**demo:** https://snip.sperm.monster

## What is snip?

snip is a small file sharing website I built for quickly uploading files and sharing them with other people.
## Features

- File uploads up to **100 MB** (FOR FREE!!!!)
- Drag and drop support
- 40+ domains to choose from
- Cool retro design (trust)
- Shareable upload links

## How to use this

```text
Choose a file → Select a domain → Upload → Copy the link
